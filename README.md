# Lista-de-Tarefas
Lista de tarefas feito com React.  Front end elabora de forma responsiva e visual.  Botões com efeitos criados usando a pseudo-classe :hover  também utilizamos &lt;input> para acrescentar dados de entrada na caixa texto.  Também foi utilizado &lt;span> para criar textos dentro de uma &lt;div>.  Todos elementos centralizado e responsivo dentro do &lt;body> .
